# 🎬 Netflix Clone Website

## 📌 Project Overview
This project is a **Netflix Landing Page Clone** built using **HTML and CSS**.  
It replicates the design and layout of the Netflix homepage, including the hero section, feature sections, FAQ section, and footer.

The goal of this project is to practice **frontend web development**, including layout design, responsive structure, and modern UI styling.

---

## ⚙️ Technologies Used

- HTML5
- CSS3

## 🖥️ Features

### 🎥 Netflix Style Landing Page
- Netflix logo and navigation bar
- Sign in and language buttons

### 📺 Hero Section
- Main headline
- Subscription message
- Email input field
- Get Started button

### 📱 Feature Sections
Includes multiple sections explaining Netflix features such as:

- Watching on TV
- Download shows to watch offline
- Kids profile support

### ❓ FAQ Section
Frequently Asked Questions section including:

- What is Netflix
- How much does Netflix cost
- Where can I watch
- How can I cancel
- What can I watch on Netflix
- Is Netflix good for kids

### 📄 Footer Section
Footer contains useful links and Netflix support information.

---

## 🎨 UI Design

The design follows the **Netflix dark theme style** including:

- Black background
- Red highlight buttons
- Large hero text
- Section separators
- Grid based footer layout
